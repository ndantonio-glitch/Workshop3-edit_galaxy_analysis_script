# Workshop3-edit_galaxy_analysis_script
This repository is for practicing a GitHub version control workflow on a python script for reading in and exploring a table of galaxy properties.
## Galaxy Analysis
## This code was created to acquire basic statistics (such as the median, mean, minimum, and maximum of a given column) of galaxies from the Portsmouth Group dataset and to plot the given data via histogram and scatterplot. 
## This code requires Python 3 or later to run.
## Packages required: Pandas, MatPlotLib, astropy.cosmology, astropy.units, and numpy
## Users must download the following datasets:
  - 'Data/Random-SF-Galaxies-PortsmouthGroup_properties.csv'
  - 'Data/Random-SB-Galaxies-PortsmouthGroup_properties.csv'
  - 'Data/Random-AGN-Galaxies-PortsmouthGroup_properties.csv'
